### Bitwise operators 

**Bitwise AND** - Returns 1 if both bits are 1 else 0
```
x & y

Example:
# 6 & 4
# 0110 & 0100
# 0100

Answer: 4  
```
<br>

**Bitwise OR** - Returns 1 if either of the bits are 1 else 0
```
x | y

Example:
# 1 | 2
# 0001 | 0010
# 0000

Answer: 0  
```
